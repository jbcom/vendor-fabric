# Agentic Workflows

`vendor_fabric.agentic` is the orchestration layer for vendor-aware agent
workflows. It keeps crew definitions framework-agnostic while letting users run
on CrewAI, LangGraph, Strands, or local CLI agents when those optional runtimes
are installed.
